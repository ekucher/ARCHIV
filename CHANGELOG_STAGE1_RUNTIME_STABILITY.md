# Stage 1 / Stage 2 — Підвищення стабільності runtime

## Гілка

stage1-runtime-stability

## Коміти

- 144b764 — Improve runtime stability safeguards
- c1249af — Release maintenance mutex before exit

## Коротко

У цьому наборі змін покращено стабільність запуску та виконання основного maintenance-скрипта BRAVO.

Основна мета — зменшити ризики, пов'язані з:

- паралельним запуском кількох екземплярів скрипта;
- некоректною передачею пароля під час відновлення архіву;
- ненадійним визначенням старих директорій через CreationTime;
- неправильною передачею масиву аргументів у зовнішні команди PowerShell;
- відсутністю явного звільнення mutex перед завершенням процесу.

## Що змінено

### 1. Захист від паралельного запуску

Додано глобальний іменований mutex:

Global\BRAVO_MAINTENANCE

Він блокує одночасний запуск кількох екземплярів BRAVO_MAINTENANCE.

Якщо інший екземпляр уже працює, скрипт завершується з кодом 2.

### 2. Явне звільнення mutex

Додано функцію:

Release-BravoMaintenanceMutex

Перед фінальним exit скрипт явно:

- звільняє mutex;
- очищає прапорець отримання mutex;
- виконує Dispose;
- скидає посилання на mutex у null.

### 3. Безпечніше відновлення архіву

У Restore-FromArchive параметр -p для 7-Zip тепер додається тільки якщо:

- режим пароля архіву увімкнений;
- ArchivePassword не порожній.

Це прибирає передачу порожнього -p для архівів без пароля.

### 4. Надійніше визначення старих директорій

У Compress-OldData старі директорії тепер визначаються за датою в імені каталогу:

YYYY-MM-DD

Замість використання файлового CreationTime.

Це надійніше, тому що CreationTime може змінюватися після копіювання, відновлення, перенесення між дисками або роботи backup/restore-інструментів.

### 5. Коректна передача аргументів у зовнішні команди

У Invoke-CommandWithLog виправлено виклик зовнішньої команди.

Було:

& Command Arguments

Стало:

& Command @Arguments

Тепер кожен елемент масиву аргументів передається як окремий аргумент.

## Перевірка

Локально виконано збірку монолітного runtime-скрипта:

Build-BRAVO-Monolith.ps1 -Clean -CreateSha512

Результат:

Build completed successfully.
Syntax OK: dist\BRAVO_MAINTENANCE.ps1
SHA512 created: dist\BRAVO_MAINTENANCE.ps1.sha512
